A) Requirements - top

This HTML5 template is a built by HTML5, CSS3, Javascript, and Jquery. This HTML5 template is running WebGL.

This HTML5 depedencies is :

  • Three.js
  • Three.js Orbit Control
  • Three.js Device Orientation
  • jQuery

This HTML5 template required webserver to run. Use XAMPP for windows or MAMP for osx or any other similar software for running local webserver, in your local development.

B) Getting Started - top

There are two other folders in this folder besides the documentation, which are called Generator and Viewer. I'm using one JSON file in this HTML5 Template. So you just need that file to edit rooms and annotations that you want to show. The point.json file is in the both Generator and Viewer folders.

Step by step

  1. Copy 360 Panorama Tour Folder to your htdocs directory
  2. Go to localhost/360 Panorama Tour/generator/
  3. Copy the 360 images to /generator/images/room and /viewer/images/room folder
  4. Copy the annotation icon images to /generator/images/icon and /viewer/images/icon folder
  5. Open /generator/point.json in text editor and edit the images and annotation object in the file
  6. Open in browser localhost/360 Panorama Tour/generator/index.html
  7. Add the annotation with left click + shift and left click + control to remove the annotation
  8. You can change the name of images, titles, descriptions in the textarea in the bottom of the page
  9. After that you can copy the text in textarea and paste it to point.json file in localhost/360 Panorama Tour/viewer/ folder

B) PHP Generator - top

In this folder, you can automatically generate point.json and /images folder. I'm using PHP for moving your image to images folder. So you can do add room/image, add annotation, and after all you just click generate button on the right bottom corner, and then the point.json file is ready automatically downloaded, after that you can copy that point.json file to viewer folder, and copy /php-generator/images/room folder to viewer/images/room folder.

Step by step

  1. Add the images

  2. Add the annotations

  3. Edit the annotations content or annotation link to other image

C) Config File - top

You just need change point.json file to edit rooms and annotations

point.json

var supportgyro = true; //if true gyroscope mode in mobile with https protocols will enabled
var DATA = [
{
	"id":1, //this id is identifier of room / image
	"name":"Kitchen", //this name is name of room / image
	"image":"./images/room/kitchen.jpg", //this url of image of room
	"annotation":[ //this annotation used by system to create annotations in the room
		 {
			pointX : -12.093831948363794, //Position 3D in X dimension, generated by generator
			pointY : -224.28607211178354, //Position 3D in X dimension, generated by generator
			pointZ : 871.5212766538773, //Position 3D in X dimension, generated by generator
			id : 1, //this id is identifier of annotation in the room
			icon : "./images/icon/icon1.png", //this url of icon of annotation
			title : "Garage", //this is used for title and label for annotation
			desc : "Description", //this description text for annotation
			link : 2 //this is id which linked to another image / room or you can use null if this annotation is description
		},
		{
			pointX : -868.6642812767091, //Position 3D in X dimension, generated by generator
			pointY : -98.12945468182242, //Position 3D in X dimension, generated by generator
			pointZ : -213.99255963043308, //Position 3D in X dimension, generated by generator
			id : 2, //this id is identifier of annotation in the room
			icon : "./images/icon/icon1.png", //this url of icon of annotation
			title : "Drawer", //this is used for title and label for annotation
			desc : "Description", //this description text for annotation
			link : null //this is id which linked to another image / room or you can use null if this annotation is description
		}			
	]
}

You can add another room with the same structure and content as much as you want.
